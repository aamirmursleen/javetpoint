Tamil nadu Pin Code

Tamil nadu Pin Code: Postal Index Number or PIN or Pin Code of Tamil nadu is a six digit number provided by Indian Post Administration.

The last three digit of Tamil nadu Pin Code is the actual digit provided to the individual post offices. The 1st digit of Tamil nadu Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

TAMIL NADU has total 31 districts and 10531 post offices in the state.
