Odisha Pin Code

Odisha Pin Code: Postal Index Number or PIN or Pin Code of Odisha is a six digit number provided by Indian Post Administration.

The last three digit of Odisha Pin Code is the actual digit provided to the individual post offices. The 1st digit of Odisha Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

ODISHA has total 30 districts and 7349 post offices in the state.
