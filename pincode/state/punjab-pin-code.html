Punjab Pin Code

Punjab Pin Code: Postal Index Number or PIN or Pin Code of Punjab is a six digit number provided by Indian Post Administration.

The last three digit of Punjab Pin Code is the actual digit provided to the individual post offices. The 1st digit of Punjab Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

PUNJAB has total 23 districts and 3589 post offices in the state.
