Kerala Pin Code

Kerala Pin Code: Postal Index Number or PIN or Pin Code of Kerala is a six digit number provided by Indian Post Administration.

The last three digit of Kerala Pin Code is the actual digit provided to the individual post offices. The 1st digit of Kerala Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

KERALA has total 14 districts and 4926 post offices in the state.
