Madhya pradesh Pin Code

Madhya pradesh Pin Code: Postal Index Number or PIN or Pin Code of Madhya pradesh is a six digit number provided by Indian Post Administration.

The last three digit of Madhya pradesh Pin Code is the actual digit provided to the individual post offices. The 1st digit of Madhya pradesh Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

MADHYA PRADESH has total 52 districts and 7329 post offices in the state.
