Nagaland Pin Code

Nagaland Pin Code: Postal Index Number or PIN or Pin Code of Nagaland is a six digit number provided by Indian Post Administration.

The last three digit of Nagaland Pin Code is the actual digit provided to the individual post offices. The 1st digit of Nagaland Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

NAGALAND has total 11 districts and 328 post offices in the state.
