Bihar Pin Code

Bihar Pin Code: Postal Index Number or PIN or Pin Code of Bihar is a six digit number provided by Indian Post Administration.

The last three digit of Bihar Pin Code is the actual digit provided to the individual post offices. The 1st digit of Bihar Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

BIHAR has total 38 districts and 7810 post offices in the state.
