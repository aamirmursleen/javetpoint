Chandigarh Pin Code

Chandigarh Pin Code: Postal Index Number or PIN or Pin Code of Chandigarh is a six digit number provided by Indian Post Administration.

The last three digit of Chandigarh Pin Code is the actual digit provided to the individual post offices. The 1st digit of Chandigarh Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

CHANDIGARH has total 1 districts and 51 post offices in the state.
