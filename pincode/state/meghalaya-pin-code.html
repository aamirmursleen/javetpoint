Meghalaya Pin Code

Meghalaya Pin Code: Postal Index Number or PIN or Pin Code of Meghalaya is a six digit number provided by Indian Post Administration.

The last three digit of Meghalaya Pin Code is the actual digit provided to the individual post offices. The 1st digit of Meghalaya Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

MEGHALAYA has total 7 districts and 468 post offices in the state.
