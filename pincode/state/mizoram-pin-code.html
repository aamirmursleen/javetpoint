Mizoram Pin Code

Mizoram Pin Code: Postal Index Number or PIN or Pin Code of Mizoram is a six digit number provided by Indian Post Administration.

The last three digit of Mizoram Pin Code is the actual digit provided to the individual post offices. The 1st digit of Mizoram Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

MIZORAM has total 8 districts and 406 post offices in the state.
