Andhra pradesh Pin Code

Andhra pradesh Pin Code: Postal Index Number or PIN or Pin Code of Andhra pradesh is a six digit number provided by Indian Post Administration.

The last three digit of Andhra pradesh Pin Code is the actual digit provided to the individual post offices. The 1st digit of Andhra pradesh Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

ANDHRA PRADESH has total 23 districts and 13557 post offices in the state.
