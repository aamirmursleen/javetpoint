Assam Pin Code

Assam Pin Code: Postal Index Number or PIN or Pin Code of Assam is a six digit number provided by Indian Post Administration.

The last three digit of Assam Pin Code is the actual digit provided to the individual post offices. The 1st digit of Assam Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

ASSAM has total 23 districts and 3841 post offices in the state.
