Karnataka Pin Code

Karnataka Pin Code: Postal Index Number or PIN or Pin Code of Karnataka is a six digit number provided by Indian Post Administration.

The last three digit of Karnataka Pin Code is the actual digit provided to the individual post offices. The 1st digit of Karnataka Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

KARNATAKA has total 30 districts and 8741 post offices in the state.
