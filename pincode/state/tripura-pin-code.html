Tripura Pin Code

Tripura Pin Code: Postal Index Number or PIN or Pin Code of Tripura is a six digit number provided by Indian Post Administration.

The last three digit of Tripura Pin Code is the actual digit provided to the individual post offices. The 1st digit of Tripura Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

TRIPURA has total 4 districts and 706 post offices in the state.
