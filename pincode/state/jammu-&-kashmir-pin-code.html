Jammu & kashmir Pin Code

Jammu & kashmir Pin Code: Postal Index Number or PIN or Pin Code of Jammu & kashmir is a six digit number provided by Indian Post Administration.

The last three digit of Jammu & kashmir Pin Code is the actual digit provided to the individual post offices. The 1st digit of Jammu & kashmir Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

JAMMU & KASHMIR has total 18 districts and 1619 post offices in the state.
