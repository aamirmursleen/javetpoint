Uttar pradesh Pin Code

Uttar pradesh Pin Code: Postal Index Number or PIN or Pin Code of Uttar pradesh is a six digit number provided by Indian Post Administration.

The last three digit of Uttar pradesh Pin Code is the actual digit provided to the individual post offices. The 1st digit of Uttar pradesh Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

UTTAR PRADESH has total 70 districts and 15304 post offices in the state.
