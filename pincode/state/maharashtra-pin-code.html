Maharashtra Pin Code

Maharashtra Pin Code: Postal Index Number or PIN or Pin Code of Maharashtra is a six digit number provided by Indian Post Administration.

The last three digit of Maharashtra Pin Code is the actual digit provided to the individual post offices. The 1st digit of Maharashtra Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

MAHARASHTRA has total 34 districts and 10546 post offices in the state.
