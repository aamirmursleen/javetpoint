Haryana Pin Code

Haryana Pin Code: Postal Index Number or PIN or Pin Code of Haryana is a six digit number provided by Indian Post Administration.

The last three digit of Haryana Pin Code is the actual digit provided to the individual post offices. The 1st digit of Haryana Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

HARYANA has total 19 districts and 2555 post offices in the state.
