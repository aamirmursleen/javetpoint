Pondicherry Pin Code

Pondicherry Pin Code: Postal Index Number or PIN or Pin Code of Pondicherry is a six digit number provided by Indian Post Administration.

The last three digit of Pondicherry Pin Code is the actual digit provided to the individual post offices. The 1st digit of Pondicherry Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

PONDICHERRY has total 3 districts and 94 post offices in the state.
