Manipur Pin Code

Manipur Pin Code: Postal Index Number or PIN or Pin Code of Manipur is a six digit number provided by Indian Post Administration.

The last three digit of Manipur Pin Code is the actual digit provided to the individual post offices. The 1st digit of Manipur Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

MANIPUR has total 9 districts and 690 post offices in the state.
