Himachal pradesh Pin Code

Himachal pradesh Pin Code: Postal Index Number or PIN or Pin Code of Himachal pradesh is a six digit number provided by Indian Post Administration.

The last three digit of Himachal pradesh Pin Code is the actual digit provided to the individual post offices. The 1st digit of Himachal pradesh Pin Code represents region, 2nd digit represents sub-region and the 3rd digit represents sorting district.

HIMACHAL PRADESH has total 12 districts and 2609 post offices in the state.
